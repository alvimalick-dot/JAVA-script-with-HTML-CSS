# Todo App

A modern, interactive todo list application with drag-and-drop functionality and theme switching capabilities.

## Features

- **Add Tasks**: Create new todo items by typing and pressing Enter
- **Mark Complete**: Click the circular checkbox to mark tasks as complete/incomplete
- **Delete Tasks**: Remove individual tasks using the cross icon
- **Drag & Drop**: Reorder tasks by dragging them to new positions
- **Filter Tasks**: View all tasks, only active tasks, or only completed tasks
- **Clear Completed**: Remove all completed tasks at once
- **Theme Toggle**: Switch between light and dark themes
- **Task Counter**: Displays the number of active (incomplete) tasks

## How to Use

1. **Adding a Task**: Type your task in the input field and press Enter
2. **Completing a Task**: Click the circle button next to a task to mark it complete
3. **Deleting a Task**: Click the × icon on the right side of a task
4. **Reordering Tasks**: Click and drag a task to move it up or down
5. **Filtering**: Use the All/Active/Complete buttons to filter your view
6. **Changing Theme**: Click the sun/moon icon in the top right corner
7. **Clear Completed**: Click "Clear Completed" to remove all finished tasks

## Installation

1. Download all files to the same directory
2. Open `task3.html` in a web browser
3. Start managing your tasks!
